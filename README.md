## Overview

Course project analyzing a single spacecraft from NASA's Starling mission — a 6U CubeSat in SSO at ~480 km, built on the BCT XB1 bus. Report and notebook grow each homework assignment.

**Current coverage (HW1):**
- 6U spacecraft model and inertia matrix (Fusion 360 CAD)
- Orbital dynamics in ECI: two-body, J₂, atmospheric drag
- Sun-synchronous orbit design and RAAN precession verification
- Torque-free Euler's equation and principal axis stability
