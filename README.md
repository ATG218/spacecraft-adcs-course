## Overview

This repository contains the simulation code and written report for Homework 1 of the course project. The selected spacecraft is NASA's **Starling** mission - a swarm of four 6U CubeSats in a Sun-synchronous orbit at ~480 km altitude, built on the Blue Canyon Technologies XB1 bus.

The work for this assignment covers:
- Spacecraft geometric model and inertia matrix (6U rectangular bus)
- Orbital dynamics ODE in ECI coordinates (two-body gravity, J2 oblateness, atmospheric drag)
- Sun-synchronous inclination derivation and RAAN precession verification
- Euler's equation simulation and principal axis stability analysis

## Running the Notebook

The notebook (`orbitode.ipynb`) requires **Julia** with the following packages:

| Package | Purpose |
|---|---|
| `DifferentialEquations` | Vern9 ODE solver for orbital integration |
| `StaticArrays` | Static vectors for drag computation |
| `MeshCat` | Interactive 3D orbit animation |
| `GeometryBasics` | Geometry primitives for MeshCat |
| `CoordinateTransformations` | Rigid-body transforms for animation |
| `Colors` | Material colors for MeshCat objects |
| `PythonPlot` | 2D/3D plots saved to `figs/` |

`LinearAlgebra` and `Statistics` are Julia standard library packages and require no installation.

Install all non-stdlib packages from the Julia REPL:

```julia
using Pkg
Pkg.add(["DifferentialEquations", "StaticArrays", "MeshCat",
         "GeometryBasics", "CoordinateTransformations", "Colors", "PythonPlot"])
```

`PythonPlot` also requires `matplotlib` in your Python environment:


The MeshCat animation opens in a browser tab automatically (`open(vis)`).
